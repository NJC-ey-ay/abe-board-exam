# ABE Study - Agricultural and Biosystems Engineering Board Exam Prep

A comprehensive study website for ABE board exam preparation based on PRC ABELE-TOS (Table of Specifications).

## Features

### Practice Questions
- Questions generated based on ABE TOS covering all subjects
- Step-by-step solutions for both correct and incorrect answers
- Weak point tracking and recommendations
- Difficulty levels (easy, medium, hard)
- Subject-based practice

### Unit Conversions
- Dedicated practice module for all engineering unit conversions
- Reference tables organized by category
- Memory techniques for effective retention
- Common conversion groups and equivalencies

### Progress Tracking
- Monitor performance across subjects
- Identify weak points requiring additional study
- Spaced repetition recommendations

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Tech Stack

- Next.js 14 (React framework)
